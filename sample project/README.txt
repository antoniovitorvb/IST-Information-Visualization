1 - To initiate the project you will need to run http server, you can do this by:

Opening the terminal in the folder Vis (right click on the opened folder -> open terminal) and run python -m http.server

2 - After running this command you will get a message like this showing in which port the http server will be running:

Serving HTTP on :: port 8000 (http://[::]:8000/) ...

3 - Open the browser and type localhost:8000 and this will open the visualization.


Environment where we developed the project:

15.6'' monitor with 100% zoom on F11 (Fullscreen) or
24 '' monitor with 125% zoom on F11 (Fullscreen).
